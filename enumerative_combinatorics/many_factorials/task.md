## @{keyword.statement}

@{lang.en}

This problem has $T$ cases.

Given a non-negative integer $N$. Print $N!$ modulo $@{param.MOD}$. 

@{lang.ja}

この問題は $T$ ケースあります．
非負整数 $N$ が与えられます．$N!$ を $@{param.MOD}$ で割った余りを出力してください．

@{lang.end}

## @{keyword.constraints}

- $1\leq T\leq @{param.T_MAX}$
- $0 \leq N < @{param.MOD}$

## @{keyword.input}

```
$T$
$N_1$
$\vdots$
$N_T$
```

## @{keyword.output}
@{lang.en}
Print $N!$ modulo $@{param.MOD}$. 

@{lang.ja}

$N!$ を $@{param.MOD}$ で割った余りを出力してください．
@{lang.end}
